Manual de instalación









      Manual de instalación                                                


 1.  Descarga del archivo .ZIP desde el repositorio de GitHub.                


Para instalar el Gestor de Alumnos debes acceder a la URL del repositorio de GitHub:  https://github.com/alexisnavarromoreno/GestorDeAlumnos


Una vez allí deberás descargar el archivo .zip del proyecto, para ello pulsaremos en el botón code y a continuación en Download ZIP.



Una vez hecho esto se nos descargará el .ZIP del proyecto, deberemos extraer los archivos en la carpeta que queramos.



 2.  Instalación de Python3 y de las librerías necesarias para ejecutar el programa.



En este momento debemos instalar en nuestro ordenador tanto las librerias utilizadas para el proyecto como el compilador de Python.


Instalación de Python3 y librerias necesarias Windows:


Puedes descargar Python para Windows desde la siguiente web https://www.python.org/downloads/windows/. Clica en el enlace "Latest Python         3 Release -Python x.x.x". Si tu ordenador ejecuta la         versión de 64 bits de Windows, descarga Windows x86-64 executable installer. De         lo contrario, descarga Windows x86 executable installer. Después de descargar el instalador, deberías         ejecutarlo (dándole doble click) y         seguir las instrucciones.

Una cosa para tener en cuenta: Durante la instalación, verás una ventana de "Setup". Asegúrate de marcar las casillas "Add Python 3.6 to         PATH" o "Add Python to your environment variables" y hacer click en "Install Now".


Una vez hecho esto debemos instalar las librerias necesarias, para ello, en la barra de tareas de Windows en su buscador escribimos cmd y         pulsamos Enter para abrir la consola de comandos.

Una vez allí para instalar las librerías necesarias deberemos introducir los siguientes comandos: 

 

        • python3-pip


        • pip install Pyside6
        • pip install sqlite3
        • pip install reportlab
        • pip install pdfrw
        • pip install pyqtgraph
        • pip install pathlib


    


Esta es la estructura del proyecto una vez descargado, para ejecutarlo tendrás que abrir el archivo Trabajo5.py.
